Resumen:

Este sitio esta hecho para crear usuarios de tipo
profesional, administrativo y clientes, en el cual se pueden
ademas, modificar y eliminar los usuarios con los campos
correspondientes a cada caso.

Cabe destacar que el desarrollo es responsivo por lo que se
ajusta su interfaz correctamente en cualquier dispositivo.
